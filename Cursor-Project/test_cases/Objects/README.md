# Objects – Entity-based test cases

In this folder you will find test cases **grouped by entity (object)**.

Each subfolder is one entity. The folder name tells you what type of tests are inside:

| Subfolder | Content |
|-----------|--------|
| **Product_contract/** | Product contract (`product_contract`) tests: Create, Edit, View, status changes, Basic Parameters, validations, etc. |
| **Service_contract/** | Service contract tests (when added). |
| **Customer/** | Customer entity tests (when added). |
| **Invoice/** | Invoice tests (when added). |
| *(add more as needed)* | |

Inside each entity folder: one or more `.md` files per **action or scenario** (e.g. `Create.md`, `Edit.md`, `View.md`, `Termination.md`).
