# /feedback

**Usage:** `/feedback`

## What this command does

1. Collects the current session's feedback content.
2. Saves it under `reports/Feedback/YYYY/<month>/<DD>/Feedback_<slug>_<HHMM>.md`.
3. Confirms the full file path in chat.

## When to use

- When you want to persist feedback about a feature, process, or finding.
- When you want to record observations for later review.
