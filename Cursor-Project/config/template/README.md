# Project templates (`config/template/`)

Canonical markdown templates for Cursor workflows and agents.

| File | Purpose |
|------|---------|
| `Test_case_template.md` | Structure and rules for test case `.md` files under `test_cases/`. |
| `Slack_report_template.md` | HandsOff Slack messages: Playwright test results to tester and #ai-report. |

References in rules and commands use path: `Cursor-Project/config/template/<filename>.md`.
