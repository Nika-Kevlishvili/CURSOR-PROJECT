## Playwright test results (summary)

Most failures are due to **API response shape**: the billing-run list endpoint returns a structure where `body.content ?? body ?? []` is not an array, causing `TypeError: runs.find is not a function` (or `.filter is not a function`). Tests assume a list of runs; the actual response may be paginated or nested differently. Fix: normalize the list (e.g. `Array.isArray(x) ? x : (x?.content && Array.isArray(x.content) ? x.content : [])`) before calling `.find`/`.filter`.
