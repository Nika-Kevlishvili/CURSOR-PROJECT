## ­ЪћЇ рЃ«рЃерЃўрЃарЃў рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃћрЃЉрЃў рЃЊрЃљ рЃњрЃљрЃЊрЃљрЃгрЃДрЃЋрЃћрЃбрЃћрЃЉрЃў

### 1. РюЁ MCP рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃўрЃА рЃцрЃљрЃўрЃџрЃўрЃА рЃЏрЃЊрЃћрЃЉрЃљрЃарЃћрЃЮрЃЉрЃљ

**рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃљ**: MCP рЃАрЃћрЃарЃЋрЃћрЃарЃћрЃЉрЃў рЃљрЃа рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА

**рЃњрЃљрЃЊрЃљрЃгрЃДрЃЋрЃћрЃбрЃљ**:
- Windows-рЃќрЃћ MCP рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА: `%APPDATA%\Cursor\mcp.json`
- рЃњрЃљрЃЊрЃљрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ рЃарЃЮрЃЏ рЃцрЃљрЃўрЃџрЃў рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА рЃљрЃЏ рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃќрЃћ

**рЃерЃћрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃљ PowerShell-рЃерЃў**:
```powershell
# рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ рЃцрЃљрЃўрЃџрЃў рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА рЃЌрЃБ рЃљрЃарЃљ
Test-Path "$env:APPDATA\Cursor\mcp.json"

# рЃЌрЃБ рЃљрЃа рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА, рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ
$mcpPath = "$env:APPDATA\Cursor\mcp.json"
if (-not (Test-Path $mcpPath)) {
    Write-Host "MCP config file not found at: $mcpPath"
    Write-Host "Please copy your mcp-config.json to this location"
}
```

### 2. РюЁ Node.js рЃЊрЃљ NPX рЃЋрЃћрЃарЃАрЃўрЃћрЃЉрЃў

**рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃљ**: NPX рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃўрЃљ, рЃЏрЃљрЃњрЃарЃљрЃЏ MCP рЃАрЃћрЃарЃЋрЃћрЃарЃћрЃЉрЃў рЃљрЃа рЃўрЃгрЃДрЃћрЃЉрЃљ

**рЃњрЃљрЃЊрЃљрЃгрЃДрЃЋрЃћрЃбрЃљ**:
- Node.js рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА v18 рЃљрЃю рЃБрЃцрЃарЃЮ рЃљрЃ«рЃљрЃџрЃў
- NPX рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА npm-рЃЌрЃљрЃю рЃћрЃарЃЌрЃљрЃЊ рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃў

**рЃерЃћрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃљ**:
```powershell
# рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ Node.js рЃЋрЃћрЃарЃАрЃўрЃљ
node --version
# рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА v18.x.x рЃљрЃю рЃБрЃцрЃарЃЮ рЃљрЃ«рЃљрЃџрЃў

# рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ npm рЃЊрЃљ npx

# рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ PATH-рЃерЃў рЃљрЃарЃўрЃА рЃЌрЃБ рЃљрЃарЃљ

where.exe npx
```

**рЃЌрЃБ рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃљрЃљ**:
```powershell
# рЃњрЃљрЃЊрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃћрЃЌ Node.js рЃЮрЃцрЃўрЃфрЃўрЃљрЃџрЃБрЃарЃў рЃАрЃљрЃўрЃбрЃўрЃЊрЃљрЃю
```

**рЃњрЃљрЃЊрЃљрЃгрЃДрЃЋрЃћрЃбрЃљ**:
- рЃЊрЃљрЃарЃгрЃЏрЃБрЃюрЃЊрЃўрЃЌ рЃарЃЮрЃЏ Node.js PATH-рЃерЃўрЃљ
- Cursor-рЃўрЃА рЃњрЃљрЃЊрЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃљ рЃерЃћрЃЏрЃЊрЃћрЃњ PATH-рЃўрЃА рЃерЃћрЃфрЃЋрЃџрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ

**рЃерЃћрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃљ**:

**Cursor-рЃўрЃА рЃњрЃљрЃЊрЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃљ**:

### 4. РюЁ MCP рЃърЃљрЃЎрЃћрЃбрЃћрЃЉрЃўрЃА рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃљ

**рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃљ**: `mcp-postgres-server` рЃърЃљрЃЎрЃћрЃбрЃў рЃљрЃа рЃЏрЃЮрЃўрЃФрЃћрЃЉрЃюрЃћрЃЉрЃљ

**рЃњрЃљрЃЊрЃљрЃгрЃДрЃЋрЃћрЃбрЃљ**:
NPX рЃърЃўрЃарЃЋрЃћрЃџ рЃњрЃљрЃерЃЋрЃћрЃЉрЃљрЃќрЃћ рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃЕрЃљрЃЏрЃЮрЃбрЃЋрЃўрЃарЃЌрЃљрЃЋрЃА рЃърЃљрЃЎрЃћрЃбрЃА, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃЊрЃарЃЮ рЃЊрЃљрЃАрЃГрЃўрЃарЃЊрЃћрЃА.

**рЃбрЃћрЃАрЃбрЃў**:

### 5. РюЁ Cursor Settings рЃЊрЃљ MCP рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ

### 8. РюЁ JSON Syntax рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃў

**рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃљ**: рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃўрЃА рЃцрЃљрЃўрЃџрЃў рЃљрЃарЃљрЃАрЃгрЃЮрЃарЃўрЃљ

**рЃњрЃљрЃЊрЃљрЃгрЃДрЃЋрЃћрЃбрЃљ**:
рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ JSON syntax:

```powershell
# рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ JSON validity
$mcpConfig = Get-Content "$env:APPDATA\Cursor\mcp.json" -Raw | ConvertFrom-Json
if ($?) {
    Write-Host "РюЁ JSON is valid"
} else {
    Write-Host "РЮї JSON syntax error!"
}
```

**рЃ«рЃерЃўрЃарЃў рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃў**:
- рЃЊрЃљрЃЎрЃљрЃарЃњрЃБрЃџрЃў рЃЏрЃФрЃўрЃЏрЃћрЃћрЃЉрЃў
- рЃЊрЃљрЃЎрЃљрЃарЃњрЃБрЃџрЃў рЃЉрЃарЃћрЃЎрЃћрЃбрЃћрЃЉрЃў `{` рЃљрЃю `}`
- рЃљрЃарЃљрЃАрЃгрЃЮрЃарЃў quotes (`"` vs `'`)

### 9. РюЁ Cursor Extension-рЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃљ

**рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃљ**: MCP extension рЃљрЃа рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃўрЃљ

**рЃњрЃљрЃЊрЃљрЃгрЃДрЃЋрЃћрЃбрЃљ**:
